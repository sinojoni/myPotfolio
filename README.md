# 📱 Portfolio Builder App

A mobile application that allows users to create and share professional developer portfolios by importing projects from GitHub or adding them manually.

---

## 🚀 Features

* 🔐 User Authentication (Create Account / Log in)
* 👤 Profile Setup (Name, Bio, Skills, Links)
* 🔗 Import Projects from GitHub
* 📄 Automatically fetch project descriptions from README files
* ✍️ Manual Project Entry with optional AI-generated descriptions
* 🖼 Upload project screenshots
* 🌐 Generate a shareable portfolio


---

## 🛠 Tech Stack

* React Native (Expo)
* Firebase (Authentication & Firestore)
* GitHub API (for fetching repositories)
* OpenAI API (for AI descriptions)

---

## 📌 Project Goals

This project aims to:

* Simplify portfolio creation for developers and students
* Reduce the need to manually build portfolio websites
* Showcase real-world projects using GitHub integration
* Provide a professional way to present skills and work
